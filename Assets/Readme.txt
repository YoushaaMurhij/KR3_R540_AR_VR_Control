Prebuilt samples for Vive Hand Tracking SDK v0.8.2.

The sample files are provided for Windows (Vive, Vive Po & Vive Cosmos) and Vive Focus. They are built using Sample scene in the Unity plugin.

For Windows demo, please follow the steps in https://developer.viveport.com/documents/sdk/en/misc/steamvr.html first.
For Focus demo, please make sure to grant camera permission at the app startup.
